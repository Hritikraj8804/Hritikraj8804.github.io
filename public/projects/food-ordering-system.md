# Food Ordering System

**Food Ordering System** is a dynamic web application that lets users explore a restaurant menu, add items to a cart, and place orders. The system provides a smooth and intuitive interface for customers to browse food items, manage their orders, and interact with backend services for real-time processing.

## Key Features

- **Menu Browsing**: Users can view available food items with images, descriptions, and prices.
- **Cart Management**: Add, update, or remove items in the cart before placing an order.
- **Order Placement**: Order submission with order summary.
- **Responsive UI**: Built with mobile-friendly design principles.
- **Backend Integration**: Uses PHP and MySQL for handling user actions and database operations.

## Project Architecture

1. **Frontend**:
   - Built with HTML, CSS, and JavaScript.
   - Provides an interactive and responsive interface.

2. **Backend**:
   - Uses PHP to manage requests, sessions, and database interactions.
   - Handles cart actions, order submissions, and data retrieval.

3. **Database**:
   - Managed through MySQL.
   - Stores menu items, cart data, and order records.

## Getting Started

### Prerequisites

- **Web Server** (e.g., XAMPP / WAMP / LAMP)
- **PHP** (version >= 7.0)
- **MySQL**
- **Browser** (for testing UI)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Hritikraj8804/food-ordering-system.git
   cd food-ordering-system
    ```

2. Move the project folder to your server‚Äôs document root:

   * XAMPP: `htdocs/`
   * WAMP: `www/`
   * LAMP: `/var/www/html/`

3. Import the database:

   * Open **phpMyAdmin**
   * Create a database (e.g., `food_ordering`)
   * Import the SQL file located in the project.

4. Update the database configuration in PHP files if required.

5. Start the web server and navigate to:

   ```
   http://localhost/food-ordering-system
   ```

### Usage

1. Open the application in your browser.
2. Browse the available food items.
3. Add desired items to the cart.
4. Review and modify your cart.
5. Place your order to complete the process.
