# EduKestra

**EduKestra** is a web-based educational platform that offers interactive tools, structured courses, and learning resources for students and learners. The platform focuses on enabling a smooth and engaging learning experience through a user-friendly interface and backend support.

## Key Features

- **Interactive Courses**: Structured learning modules with lessons and quizzes.
- **User Dashboard**: Personalized dashboard for learners to track progress.
- **Responsive UI**: Intuitive design accessible from various devices.
- **Course Enrollment**: Users can browse and enroll in available courses.
- **Backend Logic**: Flask-powered backend handles user management and content delivery.

## Project Architecture

1. **Frontend**:
   - Built with HTML, CSS, and JavaScript.
   - Provides responsive navigation and content display.

2. **Backend**:
   - Powered by Flask and Python.
   - Manages user sessions, course data, and dynamic content.

3. **Database Integration**:
   - Stores user data, course details, and progress information.
   - Connects securely through backend logic.

## Getting Started

### Prerequisites

- **Python** (version >= 3.7)
- **Flask**
- **Required Python Libraries**:
  - pandas
  - numpy
  - flask
  - SQLAlchemy (or database connector)
- **Browser** (for accessing the platform)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Hritikraj8804/EduKestra.git
   cd EduKestra
   ```

2. Create and activate a virtual environment:

   ```bash
   python -m venv env
   source env/bin/activate   # On Windows: env\Scripts\activate
   ```

3. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

4. Run the application:

   ```bash
   python app.py
   ```

5. Open your browser and go to:

   ```
   http://localhost:5000
   ```

### Usage

1. Register or log in to the platform.
2. Explore available courses and enroll.
3. View lessons, complete quizzes, and track your learning progress.
